# UrldConf (urls.py)
url을 매핑 해주는 역할을 하는 소스 파일

## urlpatterns
url을 매핑해서 View와 연결해준다.

### 형식 
``` python
urlpatterns = [
    path(경로, 뷰),
]
```
### ex)
```python
  urlpatterns = [
    path('admin/', admin.site.urls),
    path('student/', student.urls);
]
```
